# 프로젝트 이름
Account Book

## 소개
이 프로젝트는 기본적으로 인터넷 쇼핑 기능을 함과 동시에, 본인의 현재 자산 상황을 통한 적절한 상품 항목 제시 및 
구독 서비스로 인한 사용자 혜택 제공등의 기능을 담은 앱입니다  

## 주요 기능
### 프로젝트 설계.
  - 쇼핑몰 -> 기본 골자, 뼈대
	1. 소비자
		1. 기본적으로 상품/카테고리 
		2. "포인트" -> 상품 가리지 않고 해당 플랫폼에서 구매를 하면 할 수록 쌓이는 "포인트" 
		3. 목표금액을 달성했을 때 알람으로 알려주는 기능.

	  2. 상품
		1. 100만 판매 이벤트 등 알람 (감사 이벤트 - 할인쿠폰, )
		2. 상품 = 구독 느낌으로 상품을 구매한 소비자들에게만 '알람' 서비스가 가도록.
		3. 쿠팡와우 - 로켓배송/쿠팡잇츠 무료 배달/쿠팡 플레이 시청 가능.

3. 추가 기능(시간 남을 때)
	- 판매자 기능 추가
		- 감사 알람 이벤트 등을 판매자가 직접 배포
		- 100만 판매 이벤트 등 알람 (감사 이벤트 - 할인쿠폰)
	- 플래너 등 목표 금액을 보여주는 기능을 추가하기 용이함
